**Project:** Zarahemla Store API
Team Project
**Course:** CSE 341

**Project Overview:**
This video demonstrates a fully deployed REST API for an e-commerce platform hosted on **Render**.

**Key Features Implemented:**
* **Security:** GitHub OAuth Authentication protects all administrative routes (Create, Update, Delete).
* **Validation:** Robust data validation on all inputs, returning proper 400 errors for invalid data.
* **Documentation:** Complete API documentation available via Swagger UI.
* **Database:** MongoDB Atlas integration using Mongoose with complex data schemas (Products have 8+ fields).
* **Architecture:** Professional MVC structure with comprehensive global Error Handling.
